conqueso-java-client is a library to connect your Java applications to Conqueso. Everything's better... Conqueso.

Conqueso is a central repository for managing dynamic property values for the Archaius library. Refer to the project homepage for more details.

The main entry point for consumers of this library is the {@link com.rapid7.conqueso.client.ConquesoClient} class. See the documentation of that class to get started.
